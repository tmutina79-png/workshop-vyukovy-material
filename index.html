Koncept

🎵 Co je Vibe Coding?

Posun od „bušení kódu" k řízení vize.

🎼

Dirigent orchestru = Ty

Nepíšeš každou notu. Máš vizi, znáš strukturu, víš jak má výsledek znít. Říkáš AI co a proč — ne jak.

🎻

Hudebníci = AI

Copilot, Cursor, ChatGPT — to jsou tvoji hráči. Umí perfektně hrát, ale potřebují jasné instrukce (prompty).

TL;DR: Vibe coding = nemusíš umět syntaxi, ale musíš umět myslet jako designér a komunikovat s AI. Tvůj kód je konverzace.

Analýza

📐 Než začneš psát

I nejlepší AI potřebuje dobrý brief. Nejdřív mysli, pak promptuj.

🎯

1. Definuj problém

„Pro koho to je? Jaký problém to řeší?"

Příklad: „Spolužáci chtějí rychle spočítat průměr známek."

✂️

2. MVP = minimum

Minimum Viable Product — nejmenší verze, která funguje.

Neřeš login, dark mode, ani animace. Řeš jádro.

📝

3. Skica na papír

Nakresli si wireframe. Tužka + papír = nejrychlejší prototypovací tool na světě.

5 minut skicování ušetří 30 minut promptování.

🧠 Pro tip: Než otevřeš VS Code, polož si: „Kdybych měl na appku jen 1 obrazovku a 3 tlačítka — co tam bude?"

Design

🎨 Informační design

Obor, který řeší jak prezentovat data a rozhraní tak, aby jim lidé rozuměli na první pohled.

Není to „udělej to hezké" — je to „udělej to pochopitelné". Navigace v metru, dashboardy v autě, appky v mobilu — všude kolem tebe je informační design. Špatný design = uživatel odejde za 3 sekundy. Dobrý design = uživatel ani neví, že design existuje — prostě to funguje.

👁️

Hierarchie

Co vidím první?
Nadpis > podnadpis > text. Velikost fontu = důležitost. Uživatel skenuje stránku ve tvaru F (levý horní roh → doprava → dolů).

🔲

Kontrast

Je to čitelné?
Důležité prvky musí vyčnívat — barvou, velikostí, nebo prostorem kolem sebe (white-space). Malý rozdíl = neviditelný.

👆

Zpětná vazba

Reaguje to na mě?
Tlačítko se musí změnit po hoveru. Formulář musí říct „Odesláno!". Uživatel nikdy nesmí hádat, co se děje.

❌ Špatný designVšechno stejně velké, šedý text na šedém pozadí, tlačítko bez hover efektu, nulový spacing, uživatel neví kam kliknout.
✅ Dobrý designVelký nadpis nahoře, vysoký kontrast textu, tlačítko mění barvu, jasný white-space, akce je zřejmá okamžitě.

Praktická část

🏠 Projekt 1: Hypoteční kalkulačka

🟢 Low Effort — Zaměření na funkčnost
  • 🎯Cíl: Vytvořit kalkulačku, která spočítá měsíční splátku hypotéky podle ceny, úroku a doby splácení.
  • 🔑Co se naučíš: Jak AI vygeneruje kompletní logiku + vzorce. Ty jen měníš parametry a vylepšuješ design.
  • ⏱️Čas: ❓❓❓ na fungující verzi, pak vizuální vylepšení.
Vytvoř HTML stránku s hypoteční kalkulačkou. Světlý moderní design. Inputy: cena nemovitosti, vlastní prostředky, úroková sazba (%), doba splácení (roky). Výstup: měsíční splátka, celkem zaplaceno, přeplatek. Přidej posuvníky (range slidery) vedle inputů. Výsledek se přepočítá automaticky při změně. Přidej graf rozložení splátek (jistina vs. úroky). Všechno v jednom HTML souboru.

🔄 Jak to upravit? Změň na autoleasingovou kalkulačku, kalkulačku spoření, nebo třeba porovnávač tarifů. Logika výpočtů = jeden prompt.

Praktická část

📚 Projekt 2: Výukový materiál

🟠 Medium Effort — Zaměření na UI/UX
  • 🎯Cíl: Interaktivní webová stránka, která vysvětluje libovolné téma — s kvízem, animacemi a přehlednou strukturou.
  • 🔑Co se naučíš: Jak v promptu specifikovat UI pravidla — Tailwind CSS, layout, strukturu obsahu. Design je klíč.
  • ⏱️Čas: ~8 minut. Tady jde o to, aby to vypadalo profesionálně a bylo srozumitelné.
Vytvoř interaktivní výukovou webovou stránku o solární soustavě. Použij Tailwind CSS přes CDN. Světlý design. Struktura: hero sekce s nadpisem, karty pro každou planetu (název, obrázek placeholder, klíčové údaje), na konci interaktivní kvíz s 5 otázkami (multiple choice). Dodržuj pravidla informačního designu — jasná hierarchie, dobrý kontrast, zpětná vazba (správná/špatná odpověď se zvýrazní zeleně/červeně). Všechno v jednom HTML souboru.

🎨 Proč Tailwind? Protože je to „promptovatelný" — místo psaní CSS říkáš AI: „dej tomu rounded-xl, bg-white, text-lg, shadow-md" a ono rozumí. Design through words.

Praktická část

✅ Projekt 3: TO-DO List

🟠 Medium Effort — Zaměření na interaktivitu
  • 🎯Cíl: Appka na správu úkolů — přidávání, odškrtávání, mazání, filtrování. Data přežijí reload.
  • 🔑Co se naučíš: CRUD operace (Create, Read, Update, Delete) — základ každé aplikace. Plus localStorage = data zůstanou v prohlížeči.
  • ⏱️Čas: ~8 minut. Výsledek je appka, kterou můžeš fakt denně používat.
Vytvoř moderní TO-DO list aplikaci v jednom HTML souboru. Světlý čistý design, Tailwind CSS přes CDN. Funkce: přidat úkol (Enter nebo tlačítko), označit jako hotový (přeškrtne se), smazat úkol, filtrovat (všechny / aktivní / hotové). Ulož úkoly do localStorage, aby přežily reload. Přidej plynulé animace při přidání a smazání. Prázdný stav ať zobrazí emoji a motivační text.

💡 Rozšíření: Přidej priority (🔴🟡🟢), kategorie, deadline s odpočtem, nebo drag & drop řazení. Každé vylepšení = nový prompt.

Praktická část

🌐 Projekt 4: Landing Page

🔴 Vibe Master — Zaměření na deployment
  • 🎯Cíl: Profesionální one-page web s moderním designem — a nasazení na živý URL.
  • 🔑Co se naučíš: Plný workflow — prompt → kód → GitHub → live web. Kompletní pipeline.
  • ⏱️Čas: ~10 minut. Výsledek je skutečný web na internetu.
Vytvoř moderní one-page web pro můj esport tým. Použij glassmorphism design (průhledné prvky s backdrop-blur efektem). Přidej sekce: Hero s názvem týmu a mottem, O nás s kartami členů týmu, Galerie s obrázkovým gridem, a Kontaktní formulář. Použij Tailwind CSS, dark mode, gradient pozadí a smooth scroll navigaci. Všechno v jednom HTML souboru.

🚀 Deploy za 2 minuty: Push na GitHub → Settings → Pages → Deploy. Hotovo. Tvůj web je na username.github.io/repo
